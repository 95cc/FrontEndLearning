选择器

    - A.1 基础选择器
      - `tagname`：类型选择器或标签选择器。匹配目标元素的标签名。优先级：0,0,1。
      - `.class`：类选择器。匹配class属性中有指定类名的元素。优先级：0,1,0。
      - `#id`：ID选择器。匹配拥有指定ID属性的元素。优先级：1,0,0。
      - `*`：通用选择器。匹配所有元素。优先级：0,0,0。
    - A.2 组合器
        - 后代选择器（空格）：基础选择器之间有空格。例如：.nav li。
        - 子组合器（>）：匹配的目标元素是其他元素的直接后代。例如：ul > li。
        - 相邻兄弟组合器（+）：匹配的目标元素紧跟在其他元素后面。例如：li + li。
        - 通用兄弟选择器（~）：匹配所有跟随在指定元素之后的兄弟元素。注意：它不会选中目标元素之前的兄弟元素。例如：li.active ~ li。
        - `复合（compound）选择器`：
            
            多个基础选择器可以连起来（无空格或其他组合器）组成一个复合选择器。例如：`h1.page-header`。复合选择器选中的元素将匹配其全部基础选择器。例如：`.a.b`能够选中`
`，但是无法选中`
`。
            
    - A.3 伪类选择器（[MDN-伪类](https://developer.mozilla.org/zh-CN/docs/Web/CSS/Pseudo-classes)）
        
        `伪类选择器用于选中处于某个特定状态的元素。`
        
        伪类选择器始终以一个冒号(:)开始。优先级等于一个类选择器（0,1,0）。
        
        - :first-child：匹配的元素是其父元素的第一个子元素。
        - :last-child: 匹配的元素是其父元素的最后一个子元素。
        - :only-child: 匹配的元素是其父元素的唯一一个子元素（没有兄弟元素）。
        - :nth-child(an+b): 匹配的元素在兄弟元素中间有特定的位置。
            
            
            
        - :nth-last-child(an + b): 类似于:nth-child()，但是是从最后一个元素往前数。
        - :first-of-type: 根据拥有相同标签名的子元素中的数字顺序查找第一个元素。
        - :last-of-type: 匹配某种类型的最后一个子元素。
        - :only-of-type: 匹配的元素是满足该类型的唯一一个子元素。
        - :nth-of-type(an + b)
        - :nth-last-of-type(an + b)
        - :not(): 括号内的选择器必须是基础选择器。
        - :empty: 匹配的元素没有子元素（也不能有空格）。（:blank，选中仅包含空格的元素，目前没有浏览器支持）。
        - :focus: 匹配通过鼠标点击、触摸屏幕、按Tab键导航而获得焦点的元素。
        - :hover: 匹配鼠标指针正悬停在其上方的元素。
        - `:root`: 匹配文档根元素。
        - :disabled: 匹配以禁用的元素，包括input、select以及button元素。
        - :enabled
        - :checked
        - :invalid
        - :valid
        - :required: 匹配设置了required属性的元素（Level4）。
        - :optional: 匹配没有设置required属性的元素（Level4）。
    - A.4 伪元素选择器
        
        `匹配在文档中没有直接对应HTML元素的特定部分。`
        
        伪类选择器始终以两个冒号(::)开始。优先级等于一个类选择器（0,0,1）。
        
        - ::before: 创建一个伪元素，使其成为匹配元素的第一个子元素。该元素默认是行内元素，可用于插入文字、图片或其他形状。必须指定content属性才能让元素出现。例如：.menu::before。
        - ::after: 创建一个伪元素，使其成为匹配元素的最后一个子元素。该元素默认是行内元素，可用于插入文字、图片或其他形状。必须指定content属性才能让元素出现。例如：.menu::after。
        - ::first-letter: 用于指定匹配元素的第一个文本字符的样式，例如：h2::first-letter。
        - ::first-line: 用于指定匹配元素的第一行文本的样式。
        - ::selection: 用于指定用户使用鼠标高亮选择的任意文本的样式。通常用于改变选中文本的background-color。只有少数属性可以使用，包括color、background-color、cursor、text-decoration。
    - A.5 属性选择器
        
        用于根据HTML属性匹配元素。优先级同类选择（0,1,0）。
        
        - [attr]: 匹配的元素拥有指定属性attr，无论属性值是什么，例如：input[disabled]。
        - [attr=”value”]: 匹配的元素拥有指定属性attr，且属性值等于指定的字符串值，例如：input[type=”radio”]。
        - [attr^=”value”]: “开头”属性选择器。匹配元素拥有指定属性，且属性值的开头是指定的字符串值。例如：a[href^=https]。
        - [attr$=”value”]: “结尾”属性选择器。例如：a[href$=”.pdf”]。
        - [attr*=”value”]: “包含”属性选择器。例如：div[class*=”sprite-”]。
        - [attr~=”value”]: “空格分隔的列表”属性选择器。属性值是一个空格分隔的值列表，列表中的某个值等于指定的字符串值。例如：a[rel~=”author”]。
        - [attr|=”value”]: 属性值要么等于value，要么以“value-”开头。例如：div[lang|=”zh”]。
        
        **不区分大小写的属性选择器：**
        
        上面的属性选择器都区分大小写。input[value=”search”i]，可不区分大小写。但很多浏览器还不支持。
